#  Como Adicionar a Foto da Psicóloga

## Instruções:

1. **Prepare a foto:**
   - Formato recomendado: JPG ou PNG
   - Tamanho ideal: 500x500 pixels (quadrada)
   - Resolução: 72-150 dpi
   - Tamanho do arquivo: menor que 500KB

2. **Renomeie o arquivo:**
   - Nome do arquivo: psicologa.jpg ou psicologa.png

3. **Coloque o arquivo nesta pasta:**
   - Caminho: src\ClinicaPsi.Web\wwwroot\images\

4. **Atualize o index.html:**
   - Abra: src\ClinicaPsi.Web\wwwroot\index.html
   - Procure por: <div class="photo-placeholder"></div>
   - Substitua por: <img src="/images/psicologa.jpg" alt="Psicóloga Iana Santos">

## Resultado:
A foto aparecerá em formato circular na landing page com borda branca e sombra.

---
 Após adicionar a foto, faça commit e push para o Railway atualizar!
